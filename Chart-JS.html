Chart JS

Chart.js is een gratis open-source JavaScript-bibliotheek voor datavisualisatie, die 8 diagramtypen ondersteunt: staaf, lijn, gebied, cirkel (ring), bel, radar, polair en spreiding. Voor mijn chart heb ik gebruik gemaakt van de staafdiagram! 

Technische details

De staaf diagram heb ik doormiddel van onderzoek en een tutorial op YouTube gemaakt. Klik om te zien welke video ik heb gevolgd om mijn staafdiagram te maken. Verder heb ik ook mijn data gebruikt van Wikipedia. Klik ook op de link hier om mijn data bron te checken!
